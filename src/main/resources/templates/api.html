Взаимодействие с API:

Вам необходимо отправить GET (read) или POST (/delete, /update, /create) запрос на /api/{object}/method?requestParameter=x

Возможные объекты: users, departments, roles

Возможные параметры: id число

Возможные параметры: department_id число (Для user)

Возможные параметры: role_ids массив чисел (Для user)

Возможные параметры: username (Для user) name (для остальных) текст

Возможные параметры: user_ids массив чисел (Для role и department)
